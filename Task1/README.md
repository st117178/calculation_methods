# Численные методы решения нелинейных уравнений (Задание 1)

Данный программный комплекс реализует основные этапы численного решения нелинейных уравнений вида `f(x) = 0`:
1.  **Локализация (отделение) корней** на заданном отрезке методом табуляции.
2.  **Уточнение корней** четырьмя итерационными методами с заданной точностью.
3.  **Решение прикладной задачи** — определение глубины погружения шара в воду в зависимости от материала (плотности).

---

## Возможности программы

*   **Интерактивный ввод**: пользователь задает параметры задачи (границы отрезка `[A, B]`, количество разбиений `N`, точность `ε`) в диалоговом режиме.
*   **Графическое представление результатов**: все таблицы выводятся в аккуратном, читаемом формате с помощью библиотеки `tabulate`.
*   **Автоматический подбор начального приближения**: для методов Ньютона программа перебирает несколько кандидатов (концы отрезка, середину, случайную точку), чтобы повысить вероятность сходимости.
*   **Наглядное сравнение методов**: для выбранного отрезка программа запускает все четыре метода уточнения и выводит сводную таблицу, позволяя сравнить скорость сходимости (количество итераций) и точность результата.
*   **Решение физической задачи**: моделируется закон Архимеда для шара, погруженного в воду. Для каждого материала вычисляется глубина погружения.

## Структура проекта

Программа состоит из одного Python-файла, в котором определены следующие основные функции:

*   `f(x)`, `df(x)`, `d2f(x)`: Тестовая функция `10*cos(x) - 0.1*x**2` и ее производные. Могут быть заменены на другие.
*   `root_sep()`: Реализует этап отделения корней.
*   `bis_method(E, segment)`: Метод бисекций.
*   `newton_method(E, segment)`: Метод Ньютона.
*   `mod_newton_method(E, segment)`: Модифицированный метод Ньютона.
*   `secant_method(E, segment)`: Метод секущих.
*   `printer_methods(func_list, e, segment, is_print)`: Функция для запуска списка методов и вывода результатов в таблицу.
*   `ball_task()`: Реализует логику задачи о погружении шара.
*   `main`: Точка входа с главным меню.
